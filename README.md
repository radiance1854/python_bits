# python_bits

this repo contains a few modules of python that can be used as service targets for apis such as written in flask, with perhaps a react frontend calling them. the modules are meant to allow constrained management and info gathering of switches from xiq (a switch management platform) as well as creation/modification/deletion of dhcp reservations on a windows dhcp server (using powershell over ssh to connect to the box and issue the commands). it also supports basic email notification functions
